# GastroLens — Official Website

Public website for **Lens**, an AI-powered mobile app that turns dish photos into structured, professional-level ing guidance.

This repository contains the static website used for:

- Landing page
- Privacy Policy
- Terms of Use

---

## 🌿 About Lens

Lens is an AI-powered culinary app that allows users to:

- Scan a dish
- Understand its structure and essence
- Get professional-style preparation steps
- Learn chef techniques
- Improve plating and presentation

Designed for confident home recreation of restaurant-style dishes.

## 🚀 Deployment

This website is deployed as a static site using:

- Render (Static Site hosting)
- Custom domain (HTTPS enabled)

To deploy:

1. Connect this repository to Render.
2. Choose **Static Site**.
3. Leave build command empty.
4. Set publish directory to root.
5. Enable custom domain and HTTPS.
